# Security policy

Security fixes are provided for the latest released version.

Please do not disclose a suspected vulnerability in a public issue. Use
GitHub's private vulnerability reporting for this repository. Include the
affected version and platform, impact, reproduction details, and any known
workarounds. Maintainers will acknowledge a complete report as soon as
practical and coordinate disclosure after a fix is available.

Ordinary crashes, malformed geometry results, and numerical robustness issues
without a security impact should be reported through the public issue tracker.
